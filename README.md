# Heat Waves and Work-Related Accident Rates in Louisiana

A reproducible analysis investigating whether heat waves are associated with elevated rates of severe workplace injuries across 64 Louisiana parishes from 2015 to 2025.

## Overview

This notebook presents a complete analytical pipeline linking OSHA Severe Injury Reports, NOAA weather station data (GHCN-Daily), ERA5 reanalysis humidity, and U.S. Census parish-level demographics. It estimates Poisson generalized linear models with log-population offsets, year/month/day-of-week fixed effects, and parish-clustered standard errors to test the heat wave-injury association.

## Key Findings

| Analysis | Exposure | IRR | p-value |
|----------|----------|-----|---------|
| All-cause injuries (n=1,848) | Heat wave flag | 1.138 | 0.082 |
| All-cause, direct-station parishes only | Heat wave flag | 1.215 | 0.012 |
| All-cause + Parish FE | Heat wave flag | 1.151 | 0.070 |
| Heat-related injuries (n=81) | Heat wave flag | 2.256 | 0.006 |
| Outdoor industry injuries | Heat wave flag | 1.580 | <0.001 |
| Concentration test (model-free) | HW days vs non-HW | OR=10.96 | <0.0001 |

- **Continuous temperature** shows no association with all-cause injury rates (IRR = 0.997, p = 0.928)
- **Heat waves** show a suggestive 14% increase in all-cause injuries that becomes **statistically significant** (22% increase) among well-measured parishes and when restricted to heat-related outcomes (126% increase)
- **Outdoor industry workers** experience a 58% increase in injuries during heat waves
- The **concentration test** confirms heat-related incidents are 6.3x overrepresented during heat wave days

## Data Requirements

The notebook reads from pre-processed data files generated by the pipeline scripts. To reproduce from scratch, run the scripts in order:

```
scripts/01_parse_weather.py         # Parse GHCN-Daily weather data
scripts/02_fetch_era5_humidity.py   # Retrieve ERA5 reanalysis humidity
scripts/03_compute_heat_index.py    # Compute heat index and flag heat waves
scripts/04_process_census.py        # Process Census ACS demographics
scripts/05_build_master.py          # Build balanced parish-day panel
scripts/06_statistical_models.py    # Estimate Poisson GLMs and sensitivity analyses
scripts/07_descriptive_stats.py     # Generate descriptive tables
scripts/08_create_figures.py        # Generate figures
```

### Processed Data Files

The notebook expects these files under `data/processed/`:

- `master_dataset.csv` — Balanced panel (64 parishes x 3,861 days)
- `osha_processed.csv` — Geocoded OSHA severe injury reports
- `sample_construction_stats.json` — Key sample counts
- `model_diagnostics.json` — Overdispersion and AIC statistics
- `robustness_results.json` — Parish FE, labor offset, TMAX-only results
- `primary_model_coefficients.csv` — Model 1 (continuous TMAX)
- `quintile_model_results.csv` — Model 2 (temperature quintiles)
- `exploratory_hw_model_coefficients.csv` — Model 3 (heat wave flag)
- `heat_related_model_results.csv` — Heat-related outcome models
- `unified_heat_comparison_table.csv` — All-cause vs heat-related comparison
- `distance_tier_sensitivity.csv` — Interpolation distance sensitivity
- `descriptive_tables/*.csv` — Parish, industry, temporal summaries

Figures are expected under `figures/` (14 PNG files at 300 DPI).
